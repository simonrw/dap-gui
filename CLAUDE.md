# Information for agents

* **Never** consider your work finished until the code compiles (`cargo check --all-targets --all-features`) and all tests pass (`cargo xtask test && cargo xtask doctest`)
* To take a screenshot of the app, run `bin/capture_screenshot`. Screenshots should be captured to verify UX changes.
* Use [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/) for commit messages and pr titles
